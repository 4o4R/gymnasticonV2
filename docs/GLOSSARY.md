# Gymnasticon Technical Glossary

## Common Acronyms

- **API** - Application Programming Interface: A set of rules and protocols for building software
- **BLE** - Bluetooth Low Energy: A wireless protocol designed for low power devices
- **CLI** - Command Line Interface: A text-based way to interact with software
- **ESM** - ECMAScript Modules: The official standard format for packaging JavaScript code
- **NPM** - Node Package Manager: The default package manager for Node.js
- **RPi** - Raspberry Pi: A small single-board computer
- **USB** - Universal Serial Bus: A common interface for connecting devices
- **WSL** - Windows Subsystem for Linux: Allows running Linux directly on Windows
- **ANT+** - Adaptive Network Topology: A wireless sensor network protocol
- **HCI** - Host Controller Interface: The interface between Bluetooth software and hardware
- **JSON** - JavaScript Object Notation: A text format for storing and transmitting data

## Project-Specific Terms

- **Gymnasticon**: A bridge between exercise bikes and fitness apps
- **Node.js**: A JavaScript runtime that executes code outside a web browser
- **Zwift**: A virtual cycling and running platform
- **Flywheel**: A type of exercise bike
- **Peloton**: Another type of exercise bike
- **Cadence**: The rate at which a cyclist pedals (in revolutions per minute)
- **Power**: The amount of force being applied while cycling (measured in watts)

## Development Tools

- **VSCode**: Visual Studio Code - A popular code editor
- **node-gyp**: A tool for compiling native Node.js addons
- **pkg-config**: A tool for compiling and linking libraries
- **systemd**: A system and service manager for Linux
- **debugger**: A tool for finding and fixing code problems
